# MixedNotes releases

Built macOS binaries + the Sparkle appcast feed for MixedNotes.
The app source lives in a private repo; this public repo exists so
installed apps can check for and download updates.

- `appcast.xml` — Sparkle feed (the app's `SUFeedURL` points at the raw URL)
- Release assets — `MixedNotes-<version>.zip` per tagged release
